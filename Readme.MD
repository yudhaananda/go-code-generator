
# Golang Code Generator




## Run Locally

Clone the project

```bash
  git clone https://github.com/yudhaananda/go-code-generator.git
```

Go to the project directory

```bash
  cd go-code-generator
```

Build Docker App

```bash
  make docker-build
```

Run Docker App

```bash
  make docker-run
```

Run App

```bash
  make run
```
